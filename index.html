Durante o 3.º estágio no curso, tive a oportunidade de trabalhar numa empresa de desenvolvimento de software chamada Vivify, onde me foi desafiado a criar um website utilizando o Three.js, uma biblioteca para gráficos 3D. Este próprio website é mais um projeto que desenvolvi como parte de um trabalho académico, que pedia a criação de um portefólio digital. O que o torna único é que foi totalmente concebido e programado por mim, com o intuito de rever e demonstrar as minhas habilidades na criação de páginas web.
